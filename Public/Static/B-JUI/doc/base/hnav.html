横向导航菜单(hnav)

本节介绍B-JUI框架的横向导航菜单，html结构如下：

            <div id="bjui-hnav">
                <button type="button" class="bjui-hnav-toggle btn-default" data-toggle="collapse" data-target="#bjui-hnav-navbar">
                    <i class="fa fa-bars"></i>
                </button>
                <ul id="bjui-hnav-navbar">
                    
                    <li style="width:204px;"><a>欢迎您，超级管理员！</a></li>
                    
                    
                    <li class="active"><a href="javascript:;" data-toggle="slidebar">表单元素</a>
                        <ul id="bjui-hnav-tree01" class="ztree ztree_main" data-toggle="ztree" data-on-click="MainMenuClick" data-expand-all="true" data-title="表单元素演示" data-faicon="navicon" data-noinit="true">
                            <li data-id="1" data-pid="0">表单元素</li>
                            <li data-id="10" data-pid="1" data-url="form-button.html" data-tabid="form-button">按钮</li>
                        </ul>
                    </li>
                    
                    
                    <li><a href="javascript:;" data-toggle="slidebar">表单元素&演示</a>
                        <div class="trees" data-noinit="true">
                            <ul id="bjui-hnav-tree02" class="ztree ztree_main" data-toggle="ztree" data-on-click="MainMenuClick" data-expand-all="true" data-title="表单元素Demo" data-faicon="navicon">
                                <li data-id="1" data-pid="0">表单元素</li>
                                <li data-id="10" data-pid="1" data-url="form-button.html" data-tabid="form-button">按钮</li>
                            </ul>
                            <ul id="bjui-hnav-tree03" class="ztree ztree_main" data-toggle="ztree" data-on-click="MainMenuClick" data-expand-all="true" data-title="表单综合演示">
                                <li data-id="1" data-pid="0">表单演示</li>
                                <li data-id="14" data-pid="1" data-url="form.html" data-tabid="form">表单综合演示</li>
                            </ul>
                        </div>
                    </li>
                    
                    
                    <li><a href="#" class="dropdown-toggle" data-toggle="dropdown"></a>
                        <ul class="dropdown-menu" role="menu">
                            <li><a href="#"></a></li>
                            <li><a href="#"></a></li>
                            ...
                        </ul>
                    </li>
                    ...
                </ul>
            </div>

说明：

  • 横向菜单(#bjui-hnav-navbar)的li元素添加Classactive，并且是zTree菜单，则在框架初始化完成后，对应的zTree会自动加载到左侧导航栏
  • zTree菜单：横向菜单的a链接属性 data-toggle为slidebar时，点击本链接将对应的zTree加载到左侧导航栏
  • 下拉菜单：横向菜单的a链接属性 data-toggle为dropdown时，触发下拉菜单
  • zTree菜单如果要触发多棵树，需要将多个zTree(ul)放入容器div(class="trees")中，并为该容器添加属性data-noinit="true"。如果仅触发一棵树，不需要放入div容器，但需要为zTree(ul)添加属性data-noinit="true"。
  • zTree(ul)的属性data-faicon可以定义左侧导航栏模块的icon图标，属性data-faicon-close可定义模块折叠时的icon图标，属性data-title可定义模块的名称
